# 历史朋友圈生成器

## 简介

**历史朋友圈生成器** 是一个让历史人物"活起来"的创意教育Skill。如果历史人物有朋友圈，他们会发什么？通过模拟历史人物的社交动态，让学生在"刷朋友圈"中学习历史，寓教于乐。

## 核心价值

### 解决的痛点
- **历史人物太遥远**：学生觉得历史人物只是课本上的名字
- **历史事件太枯燥**：年代、地点、人物关系难以记忆
- **缺乏学习兴趣**：传统历史教学方式难以激发学生兴趣

### 创新亮点
1. **穿越式学习**：让学生"进入"历史人物的社交圈
2. **情境化记忆**：通过有趣的朋友圈内容记忆历史事件
3. **互动式体验**：可以"评论"历史动态，加深理解
4. **社交化传播**：生成的内容适合分享，寓教于乐

## 功能矩阵

| 功能 | 输入 | 输出 | 适用场景 |
|------|------|------|----------|
| 人物动态生成 | 历史人物+事件 | 朋友圈截图 | 历史人物介绍 |
| 事件现场播报 | 历史事件 | 现场照片+文案 | 重大历史事件 |
| 跨时空对话 | 多个历史人物 | 评论互动截图 | 人物关系教学 |
| 历史人物群聊 | 历史时期主题 | 群聊记录截图 | 时代背景讲解 |

## 触发条件

### 正向触发（应该调用）
- 用户提到历史人物名字（如"李白"、"秦始皇"、"曹操"）
- 用户提到历史事件（如"赤壁之战"、"贞观之治"）
- 用户说"生成历史朋友圈"、"历史人物朋友圈"
- 用户想了解历史人物的性格、思想

### 负向触发（不应调用）
- 用户只是问历史年代（简单问答）
- 用户要写历史论文（应提供学术资料）
- 用户要考试复习大纲（应提供知识点总结）

## 执行步骤

### Step 1: 人物与事件识别
解析用户输入，识别：
- 历史人物（姓名、朝代、身份）
- 历史事件（时间、地点、背景）
- 人物性格特点（豪放、内敛、幽默等）
- 历史时期特征（语言风格、服饰、生活方式）

### Step 2: 朋友圈内容创作
根据历史背景创作朋友圈内容：
- **文案风格**：结合人物性格和历史语境
- **配图描述**：符合历史时代特征
- **评论区**：其他历史人物的互动评论
- **点赞列表**：同时代或相关历史人物

### Step 3: 图像生成
调用Wan 2.7 API生成：
- **人物头像**：符合历史形象的头像
- **朋友圈配图**：历史场景、事件画面
- **朋友圈截图**：整合成完整的朋友圈界面

### Step 4: 结果输出
- 生成完整的朋友圈截图
- 附上历史知识点讲解
- 提供人物关系图谱

### Step 5: 互动扩展
- 用户可以"评论"历史动态
- 生成历史人物的"回复"
- 扩展相关历史知识

## 使用示例

### 示例1：诗人的朋友圈
```
用户：生成李白的的朋友圈
执行：
1. 识别人物：李白（唐代诗人，豪放不羁）
2. 创作内容：
   - 文案："今天又喝醉了，月亮真圆，写了一首《静夜思》，各位看看如何？"
   - 配图：月光下的诗人剪影
   - 评论：
     * 杜甫："兄台这诗，绝了！但少喝点酒啊"
     * 王维："画面感太强了，佩服"
     * 高适："又是一首千古名篇诞生了"
3. 调用wan2.7-image-pro生成图片
4. 输出：李白朋友圈截图
```

### 示例2：帝王的炫耀
```
用户：秦始皇统一六国后发的朋友圈
执行：
1. 识别人物：秦始皇（首位皇帝，雄才大略）
2. 创作内容：
   - 文案："十年磨一剑，六国尽归我大秦！统一度量衡、文字、货币，基建狂魔上线💪 #千古一帝"
   - 配图：六国地图变秦朝版图动画
   - 评论：
     * 李斯："陛下英明神武，万世基业"
     * 赵高：（点赞）
     * 孟姜女："你家墙修得太长了..."
3. 生成朋友圈截图
```

### 示例3：跨时空对话
```
用户：曹操和刘备的评论区互怼
执行：
1. 识别人物：曹操（奸雄）、刘备（仁君）
2. 创作内容：
   曹操发动态："今天煮酒论英雄，天下英雄唯使君与操耳"
   刘备评论："孟德兄过奖了，我只想种地"
   曹操回复："别装了，我懂你"
   诸葛亮评论："主公，赶紧撤，这是套路！"
3. 生成带评论互动的朋友圈截图
```

### 示例4：历史群聊
```
用户：唐朝诗人群聊
执行：
1. 生成群聊界面
2. 群成员：李白、杜甫、王维、白居易、李商隐
3. 聊天内容：
   李白：@所有人 今天谁有空陪我喝酒写诗？
   杜甫：李兄，我最近有点忙，在写"三吏三别"
   王维：我在山中参禅，你们玩
   白居易：我刚写完《长恨歌》，好累
   李商隐：各位前辈好，新人报道
4. 输出：群聊截图
```

## 配置说明

### 必需配置
```bash
export WAN_API_KEY="your-api-key"
export WAN_BASE_URL="https://dashscope.aliyuncs.com/api/v1/services/aigc/multimodal-generation/generation"
```

### 可选配置
```bash
export HISTORY_OUTPUT_DIR="./history_moments"  # 输出目录
export DEFAULT_DYNASTY="唐朝"  # 默认朝代风格
```

## 模板库

### 文案风格模板
- **诗人风格**：风雅、豪放、多愁善感
- **帝王风格**：霸气、官方、政治化
- **将军风格**：豪迈、直率、军旅气
- **才女风格**：婉约、细腻、感性
- **改革家风格**：激进、忧国忧民

### 评论风格模板
- **好友互动**：调侃、鼓励、捧场
- **政治对手**：阴阳怪气、明争暗斗
- **下属回复**：马屁、汇报、请示
- **后人点评**：历史评价、典故引用

## 失败处理

### 历史人物无法识别
- 提示用户该人物可能不够知名
- 推荐相似的历史人物

### 历史事件时间冲突
- 提示用户时间线问题
- 自动调整到合理的时间点

### 图片生成不满意
- 提供风格选择（写实/卡通/水墨）
- 支持重新生成

## 教育价值

- 历史人物生平与性格

- 😄 **趣味性**：学生愿意主动阅读
- 🧠 **记忆性**：通过有趣场景加深记忆

- 💬 **互动性**：可以"参与"历史对话
