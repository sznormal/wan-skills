# 历史朋友圈生成器

> **参赛项目**：WaytoAGI整活计划第十二期「万相妙思+」快闪赛
> **主题**：「万相皆可Skill」
> **开发者**：Normal老师团队

## 🎯 项目简介

**历史朋友圈生成器** 是一个让历史人物"活起来"的创意教育Skill。如果历史人物有朋友圈，他们会发什么？通过模拟历史人物的社交动态，让学生在"刷朋友圈"中学习历史，寓教于乐。

**核心创意**：穿越时空，进入历史人物的社交圈！

## ✨ 创新亮点

### 1. 🎭 沉浸式历史体验
- 李白发朋友圈吐槽写不出诗
- 秦始皇炫耀统一六国
- 曹操和刘备在评论区互怼
- 学生通过"刷朋友圈"学习历史

### 2. 📱 现代化表达
- 用现代社交媒体语言重新诠释历史
- 历史人物变得"有血有肉"
- 枯燥的历史事件变成有趣的社交动态

### 3. 🎨 多模态输出
- **人物头像**：符合历史形象的AI头像
- **朋友圈配图**：历史场景、事件画面
- **完整截图**：朋友圈界面模拟

### 4. 📚 知识点嵌入
- 文案中融入历史名言
- 评论中展现人物关系
- 自动提取知识点讲解

## 🔧 技术实现

### 核心技术栈
- **模型**：阿里通义万相 Wan 2.7
  - `wan2.7-image-pro`：生成人物头像、场景配图

### 关键能力利用
| Wan 2.7 能力 | 本项目应用 |
|-------------|-----------|
| 图像生成 | 生成历史人物头像、场景配图 |
| 风格控制 | 水墨风、写实风、卡通风 |
| 文字渲染 | 生成带文字的朋友圈截图 |

## 📖 使用指南

### 环境配置
```bash
# 设置API Key
export WAN_API_KEY="your-api-key"

# 安装依赖
pip install requests
```

### 快速开始
```bash
# 查看可用历史人物
python scripts/generate.py --list

# 生成李白的朋友圈
python scripts/generate.py --person 李白

# 生成特定事件的朋友圈
python scripts/generate.py --person 秦始皇 --event 统一六国
```

### 智能体调用示例
```
用户：生成李白的朋友圈
触发：历史朋友圈生成器
执行：
1. 识别人物：李白（唐代诗人，豪放不羁）
2. 生成内容：
   - 文案："今天又喝醉了，月亮真圆，写了一首《静夜思》"
   - 配图：月下独酌的水墨画
   - 评论：杜甫"兄台这诗，绝了！"
3. 调用wan2.7-image-pro生成图像
输出：李白朋友圈截图 + 知识点讲解
```
