# 科学梗图生成器

> **参赛项目**：WaytoAGI整活计划第十二期「万相妙思+」快闪赛
> **主题**：「万相皆可Skill」
> **开发者**：Normal老师团队

## 🎯 项目简介

**科学梗图生成器** 是一个把科学知识变成爆笑梗图的创意教育Skill。用网络流行的方式表达科学概念，让学习变得有趣，适合社交媒体传播，寓教于乐。

**核心理念**：科学知识 + 网络梗图 = 爆笑学习

## ✨ 创新亮点

### 1. 🎭 梗图式学习
- "当你终于理解相对论时" 😂
- "物理老师眼中的我 vs 我眼中的物理" 🤣
- "理想中的实验结果 vs 实际结果" 😭

### 2. 📱 病毒式传播
- 学生愿意主动分享到朋友圈/微博/小红书
- 每张梗图都是一个知识点
- 在转发中实现知识传播

### 3. 🎨 多模态输出
- 表情包风格
- 日漫风格
- 卡通风格
- 写实风格

### 4. 📚 知识点嵌入
- 物理概念（相对论、量子力学、牛顿定律）
- 化学知识（化学键、氧化还原、化学平衡）
- 生物知识（DNA复制、自然选择、光合作用）
- 数学概念（微积分、概率论、线性代数）

## 🔧 技术实现

### 核心技术栈
- **模型**：阿里通义万相 Wan 2.7
  - `wan2.7-image-pro`：生成梗图图像
  - 文字渲染能力：添加中文梗图文案

### 关键能力利用
| Wan 2.7 能力 | 本项目应用 |
|-------------|-----------|
| 图像生成 | 生成梗图配图、表情包 |
| 文字渲染 | 添加中文梗图文案 |
| 风格控制 | 卡通/日漫/表情包/写实 |
| 对比构图 | 左右对比、上下对比、三连 |

## 📖 使用指南

### 环境配置
```bash
# 设置API Key
export WAN_API_KEY="your-api-key"

# 安装依赖
pip install requests
```

### 快速开始
```bash
# 查看可用的科学概念
python scripts/generate.py --list-concepts

# 查看梗图类型
python scripts/generate.py --list-types

# 生成相对论梗图
python scripts/generate.py --concept "相对论"

# 指定梗图类型
python scripts/generate.py --concept "牛顿定律" --type "老师眼中的我"

# 指定风格
python scripts/generate.py --concept "量子力学" --style manga
```

### 智能体调用示例
```
用户：生成一个"当你终于理解相对论时"的梗图
触发：科学梗图生成器
执行：
1. 识别概念：相对论（物理）
2. 选择模板：当你终于理解时
3. 生成内容：
   - 上格：困惑表情 + "当你第一次听相对论时"
   - 下格：顿悟表情 + "当你终于理解相对论时"
4. 调用wan2.7-image-pro生成
输出：梗图 + 知识点注释 + 分享文案
```
