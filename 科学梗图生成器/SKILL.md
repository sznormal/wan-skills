# 科学梗图生成器

## 简介

**科学梗图生成器** 是一个把科学知识变成爆笑梗图的创意教育Skill。用网络流行的方式表达科学概念，让学习变得有趣，适合社交媒体传播，寓教于乐。

## 核心价值

### 解决的痛点
- **科学概念太抽象**：学生觉得物理、化学、生物难以理解
- **学习过程太枯燥**：传统教学方式难以激发兴趣
- **知识难以传播**：学生不愿意主动分享学习内容

### 创新亮点
1. **梗图式学习**：用网络流行梗表达科学知识
2. **病毒式传播**：学生愿意主动分享到社交媒体
3. **对比式理解**：通过反差对比加深记忆
4. **幽默化表达**：让严肃的科学变得有趣

## 梗图类型矩阵

| 类型 | 格式 | 示例 |
|------|------|------|
| **当你终于理解时** | 表情包对比 | "当你终于理解相对论时" vs "当你还在困惑时" |
| **老师眼中的我** | 上下对比 | "物理老师眼中的我" vs "我眼中的物理" |
| **理想vs现实** | 左右对比 | "理想中的实验结果" vs "实际的实验结果" |
| **科学家说vs我听到** | 对话框 | "牛顿说：F=ma" vs "我听到：艾弗等于妈" |
| **学霸vs学渣** | 人物对比 | "学霸做题时" vs "我做题时" |
| **学科拟人化** | 角色设计 | "物理是一个高冷男神" vs "化学是一个暴躁少女" |
| **知识点的日常** | 生活场景 | "电子在原子里的日常" |
| **考试状态** | 表情包 | "考试前" vs "考试时" vs "考试后" |

## 触发条件

### 正向触发（应该调用）
- 用户提到科学概念（如"相对论"、"量子力学"、"化学反应"）
- 用户说"生成梗图"、"科学梗图"
- 用户提到"当你...时"、"老师眼中的"等梗图格式
- 用户想要有趣的科学教学内容

### 负向触发（不应调用）
- 用户要严肃的科学讲解
- 用户要考试复习资料
- 用户要学术论文写作

## 执行步骤

### Step 1: 梗图类型识别
解析用户输入，识别：
- 科学概念（物理/化学/生物/数学）
- 梗图类型（当你...时、老师眼中的、理想vs现实等）
- 表达风格（表情包、对话、对比图等）

### Step 2: 内容创作
根据梗图类型创作内容：
- 选择合适的梗图模板
- 设计对比场景
- 添加搞笑文案

### Step 3: 提示词构建
为Wan API构建图像生成提示词：
- 风格：表情包风格、卡通风格、漫画风格
- 构图：上下对比、左右对比、四格漫画
- 文字：中文梗图文案

### Step 4: 图像生成
调用Wan 2.7 API：
- 生成梗图配图
- 利用文字渲染能力添加文案
- 支持多种输出格式

### Step 5: 结果输出
- 返回梗图图片
- 附上科学知识点
- 提供分享文案

## 梗图模板库

### 模板1：当你终于理解时
```
上格：【困惑状态】
- 表情：迷茫、抓狂、崩溃
- 场景：面对难题时的挣扎
- 文案："当你第一次听相对论时"

下格：【顿悟状态】
- 表情：恍然大悟、自信满满
- 场景：理解后的通透
- 文案："当你终于理解相对论时"

知识点注释：相对论核心概念
```

### 模板2：老师眼中的我
```
左格：【老师的期望】
- 形象：专注听讲、认真笔记
- 文案："物理老师眼中的我"

右格：【实际的我】
- 形象：发呆、玩手机、神游
- 文案："我眼中的物理课"

知识点注释：课堂重点内容
```

### 模板3：理想vs现实
```
左格：【理想状态】
- 实验结果完美
- 曲线平滑优美
- 文案："理想中的实验结果"

右格：【现实状态】
- 数据乱七八糟
- 曲线歪七扭八
- 文案："实际的实验结果"

知识点注释：实验误差来源
```

### 模板4：学霸vs学渣
```
左格：【学霸状态】
- 轻松自信
- 一眼看出答案
- 文案："学霸做题时"

右格：【学渣状态】
- 抓耳挠腮
- 完全看不懂
- 文案："我做题时"

知识点注释：解题技巧
```

### 模板5：学科拟人化
```
学科角色设计：
- 物理：高冷男神，话少但句句真理
- 化学：暴躁少女，一言不合就爆炸
- 生物：温柔学姐，关心生命万物
- 数学：神秘智者，只有少数人能懂

场景：学科之间的日常互动
```

### 模板6：微观世界的日常
```
场景：原子内部的日常
- 电子：到处乱跑的熊孩子
- 质子：稳如泰山的家长
- 中子：躺平的咸鱼

趣味对话和互动
```

## 使用示例

### 示例1：当你终于理解相对论时
```
用户：生成"当你终于理解相对论时"梗图
执行：
1. 识别类型：当你...时（对比型）
2. 创作内容：
   上格：困惑的学霸脸 + "时间和空间是绝对的？"
   下格：顿悟脸 + "原来时间和空间可以弯曲！"
3. 调用wan2.7-image-pro生成
4. 输出梗图 + 知识点：相对论核心概念
```

### 示例2：物理老师眼中的我
```
用户：生成"物理老师眼中的我 vs 我眼中的物理"
执行：
1. 识别类型：老师眼中的（对比型）
2. 创作内容：
   左：认真听课的学生 + "老师期望：专注听讲"
   右：发呆的学生 + "实际：在想中午吃什么"
3. 生成梗图
4. 知识点：牛顿运动定律
```

### 示例3：化学反应拟人化
```
用户：生成化学元素的反应梗图
执行：
1. 识别类型：拟人化
2. 创作内容：
   氢气："我很活泼~"
   氧气："我也是~"
   点火后：爆炸表情包
   文案："当氢气遇到氧气（有火星）"
3. 生成萌系梗图
4. 知识点：化学反应方程式
```

### 示例4：考试三连
```
用户：生成考试前中后对比梗图
执行：
1. 识别类型：三连对比
2. 创作内容：
   考前：自信满满 "稳了！"
   考中：崩溃 "这是啥？"
   考后：躺平 "算了，明年再来"
3. 生成三格梗图
```

## 配置说明

### 必需配置
```bash
export WAN_API_KEY="your-api-key"
export WAN_BASE_URL="https://dashscope.aliyuncs.com/api/v1/services/aigc/multimodal-generation/generation"
```

### 可选配置
```bash
export MEME_OUTPUT_DIR="./science_memes"  # 输出目录
export DEFAULT_STYLE="cartoon"  # 默认风格：cartoon/manga/emoji
```

## 风格选择

| 风格 | 特点 | 适用场景 |
|------|------|----------|
| **cartoon** | 卡通可爱风 | 日常科普、低年级学生 |
| **manga** | 日漫夸张风 | 理科生群体 |
| **emoji** | 表情包风 | 社交媒体传播 |
| **realistic** | 写实风 | 高年级学生 |

## 失败处理

### 科学概念无法识别
- 推荐相似的科学概念
- 提供常见科学概念列表

### 梗图类型不明确
- 推荐默认的梗图模板
- 列出可用的梗图类型

### 图像生成不满意
- 提供风格选择
- 支持重新生成

## 社交媒体适配

### 输出格式
- **朋友圈**：正方形 1:1
- **微博**：横版 16:9
- **小红书**：竖版 3:4
- **抖音/视频**：竖版 9:16

### 分享文案
自动生成适合社交媒体的分享文案，包含：
- 梗图主题
- 知识点标签
- 话题标签

- 物理：力学、电学、热学、光学、现代物理
- 化学：元素、反应、有机化学、化学键
- 生物：细胞、遗传、进化、生态
- 数学：代数、几何、微积分、概率

### 学习效果
- 😂 **趣味性**：学生愿意主动观看和分享
- 🧠 **记忆性**：通过幽默场景加深记忆
- 📱 **传播性**：适合社交媒体传播，形成学习社区
- 🔄 **复用性**：模板可反复使用，生成不同内容的梗图
