# 抽象概念具象化器 (Abstract Concept Visualizer)

## Skill 概述

**一句话描述**：把任何抽象概念变成生动的生活场景图像

**核心能力**：概念 → 比喻 → 可视化

**适用场景**：
- 教师讲解新概念时的课堂导入
- 学生理解抽象知识点的辅助材料
- 技术文档和教程的可视化配图
- 科普内容的社交媒体传播

---

## 功能特性

### 1. 智能概念映射
自动识别输入概念的类别，匹配最合适的生活比喻：

| 概念类别 | 比喻风格 |
|---------|---------|
| 数据结构 | 建筑/容器 |
| 通信协议 | 交通/物流 |
| 控制流程 | 日常生活流程 |
| 硬件接口 | 开关/水龙头 |
| 错误类型 | 意外场景 |

### 2. 多风格输出
- **卡通风格**：适合课堂教学
- **科技风格**：适合技术文档
- **手绘风格**：适合笔记和课件
- **漫画风格**：适合社交媒体

### 3. 知识点嵌入
每张图片自动附带：
- 核心概念解释
- 生活比喻说明
- 学习要点提示

## 使用方式

### 输入格式
```
概念名称 [可选:风格] [可选:比喻偏好]
```

### 示例

**基础用法**
```
用户：指针
输出：导航APP场景图 + 比喻说明

用户：GPIO
输出：开关/水龙头场景图 + 比喻说明

用户：中断
输出：门铃/紧急电话场景图 + 比喻说明
```

**指定风格**
```
用户：指针 卡通风格
用户：I2C 科技风格
用户：PWM 手绘风格
```

**指定比喻**
```
用户：指针 用门牌号比喻
用户：GPIO 用水龙头比喻
用户：中断 用门铃比喻
```

## 技术实现

### Wan 2.7 能力利用

| 能力 | 应用场景 |
|------|---------|
| 文生图 (wan2.7-image-pro) | 生成比喻场景图像 |
| 文字渲染 | 在图中添加概念名称和说明 |
| 风格控制 | 支持多种视觉风格 |
| 中文渲染 | 支持中文概念和说明 |

### API 调用示例
```python
import requests

def visualize_concept(concept, style="cartoon"):
    prompt = build_prompt(concept, style)
    
    response = requests.post(
        "https://dashscope.aliyuncs.com/api/v1/services/aigc/text2image/image-synthesis",
        headers={"Authorization": f"Bearer {API_KEY}"},
        json={
            "model": "wan2.7-image-pro",
            "input": {"prompt": prompt},
            "parameters": {"style": style}
        }
    )
    
    return response.json()
```

## 输出格式

### 标准输出
```
┌─────────────────────────────────┐
│         [生成的可视化图像]        │
│                                 │
│    💡 概念：指针                  │
│    🔗 比喻：导航APP               │
│    📖 说明：指针就像导航APP，      │
│       它不存储数据本身，           │
│       而是告诉你数据在哪里         │
└─────────────────────────────────┘
```

### 教学辅助信息
```
📌 核心要点：
- 指针存储的是地址，不是值
- 解引用 = 到达目的地
- 空指针 = 导航到虚无

⚠️ 常见误区：
- 混淆指针和指针指向的值
- 忘记初始化指针
- 野指针访问
```

## 扩展性

### 自定义概念映射
用户可以添加自定义概念：
```json
{
  "concept": "自定义概念",
  "category": "类别",
  "metaphor": "比喻场景",
  "description": "概念说明"
}
```

### 批量生成
支持批量生成多个概念的可视化：
```bash
python generate.py --batch concepts.txt --style cartoon
```
