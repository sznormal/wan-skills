# 🎨 抽象概念具象化器

> **参赛项目**：WaytoAGI整活计划第十二期「万相妙思+」快闪赛
> **主题**：「万相皆可Skill」
> **开发者**：Normal老师团队

## 🎯 项目简介

**抽象概念具象化器** 是一款专为教育工作者设计的智能视觉创作工具，能够将任何抽象的技术概念一键转化为生动的生活场景图像，让抽象知识变得看得见、摸得着。

**核心理念**：概念 → 比喻 → 可视化

---

## ✨ 创新亮点

### 1. 🧠 智能概念映射
自动识别概念类别，匹配最合适的生活比喻：

| 概念 | 比喻 | 可视化场景 |
|------|------|-----------|
| 指针 | 导航APP | 手机导航到目的地 |
| GPIO | 水龙头 | 开关控制水流 |
| 中断 | 门铃 | 有人按门铃打断工作 |
| I2C | 邮件系统 | 主从设备收发邮件 |
| PWM | 调光灯 | 旋钮调节亮度 |
| 栈溢出 | 杯子溢水 | 水杯倒水溢出 |

### 2. 🎭 多风格输出
- **卡通风格**：适合课堂教学，亲切可爱
- **科技风格**：适合技术文档，专业现代
- **手绘风格**：适合笔记课件，自然亲切
- **漫画风格**：适合社交传播，生动有趣

### 3. 📚 知识点嵌入
每张图片自动附带：
- 核心概念解释
- 生活比喻说明
- 学习要点提示
- 常见误区警示

### 4. 🔄 通用性强
- 嵌入式系统概念
- 操作系统概念
- 机器人ROS概念
- 支持自定义概念扩展

## 🔧 技术实现

### 核心技术栈
- **模型**：阿里通义万相 Wan 2.7
  - `wan2.7-image-pro`：高质量图像生成
  - 中文文字渲染能力

### 关键能力利用
| Wan 2.7 能力 | 本项目应用 |
|-------------|-----------|
| 图像生成 | 生成生活场景比喻图 |
| 文字渲染 | 添加中文概念和说明 |
| 风格控制 | 卡通/科技/手绘/漫画 |
| 智能构图 | 自动匹配场景元素 |

### 项目结构
```
抽象概念具象化器/
├── SKILL.md              # Skill定义文件
├── README.md             # 本文档
├── scripts/
│   ├── generate.py       # 核心生成脚本
│   └── batch.py          # 批量生成脚本
└── references/
    ├── concept-mapping.json  # 概念映射库
    └── style-templates.json  # 风格模板库
```

## 📖 使用指南

### 环境配置
```bash
# 设置API Key
export WAN_API_KEY="your-api-key"

# 安装依赖
pip install requests
```

### 快速开始
```bash
# 生成指针概念的可视化
python scripts/generate.py --concept "指针"

# 指定风格
python scripts/generate.py --concept "GPIO" --style cartoon

# 指定比喻
python scripts/generate.py --concept "中断" --metaphor "门铃"

# 批量生成
python scripts/generate.py --batch concepts.txt
```

### 智能体调用示例
```
用户：帮我可视化"指针"这个概念
触发：抽象概念具象化器
执行：
1. 识别概念类别：数据结构
2. 匹配比喻：导航APP
3. 构建Prompt：生成手机导航场景
4. 调用wan2.7-image-pro生成
输出：导航场景图 + 比喻说明 + 知识点
```
