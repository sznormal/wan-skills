## 🚀 快速使用

### 1. 安装依赖
```bash
cd 万相教学助手
pip install -r requirements.txt
```

### 2. 配置API密钥
```bash
export DASHSCOPE_API_KEY="your_api_key"
```

### 3. 运行示例
```bash
python examples.py
```
