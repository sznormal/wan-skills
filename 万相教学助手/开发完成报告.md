## 🚀 使用方式

### 快速开始
```python
from main import WanxTeachingAssistant

assistant = WanxTeachingAssistant(api_key="your_key")
result = assistant.generate_lesson_video(
    subject="生物",
    topic="细胞分裂",
    grade="高中",
    duration=30
)
```

### 命令行使用
```bash
python main.py --subject 生物 --topic 细胞分裂 --duration 30
```

### 打包参赛
```bash
python package.py
```
