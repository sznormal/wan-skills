# 万相教学助手 - 知识点动态可视化工具

## Skill概述

本Skill专为教师设计，利用阿里通义万相Wan 2.7的强大能力，将抽象的教学知识点转化为生动的动态视频素材。支持"知识点→故事板→视频"的完整工作流，帮助教师快速创建高质量的教学可视化内容。

## 核心价值

- **降本增效**：无需专业视频制作技能，5分钟内生成教学视频素材
- **知识可视化**：将抽象概念转化为直观的动态图像，提升学生理解力
- **可重复使用**：同一知识点可生成多种风格的教学素材
- **教学场景全覆盖**：支持科学实验、历史事件、文学场景、数学概念等多种学科

## 适用场景

1. **概念讲解**：将抽象概念（如"细胞分裂""光合作用"）转化为动画
2. **历史重现**：生成历史事件的可视化场景
3. **文学场景**：将文学作品中的描述转化为画面
4. **科学实验**：模拟实验过程的动态演示
5. **知识点复习**：生成复习用的记忆卡片式视频

## 技术架构

### API调用
- **文生图**：wan2.7-image-pro（生成教学场景图）
- **文生视频**：wan2.7-t2v（生成动态教学视频）
- **图生视频**：wan2.7-i2v（将教学插图动画化）

### 核心功能模块

#### 1. 智能提示词生成器
根据学科和知识点，自动优化提示词
```
输入：高中生物-有丝分裂
输出：专业级教学提示词，包含教育术语和视觉指导
```

#### 2. 教学风格适配
支持多种教学视频风格：
- 知识动画风（简洁扁平，适合概念讲解）
- 科幻科技风（适合物理、化学）
- 历史复古风（适合历史、文学）
- 自然写实风（适合生物、地理）

#### 3. 故事板生成器
自动生成视频分镜脚本
- 场景1：概念引入（5秒）
- 场景2：过程演示（10秒）
- 场景3：关键点强调（5秒）
- 场景4：总结回顾（5秒）

## 使用流程

### Step 1: 创建任务
```
用户输入：
- 学科：生物
- 年级：高中
- 知识点：有丝分裂的过程
- 风格：知识动画风
- 时长：25秒
```

### Step 2: 生成故事板
系统自动生成4个场景的分镜描述和提示词

### Step 3: 生成视频
- 调用wan2.7-t2v生成动态视频
- 支持多场景拼接
- 可调整节奏和转场

### Step 4: 交付成果
- 高清视频文件（MP4格式）
- 配套脚本文档
- 教学使用建议

## 使用示例

```python
from wanx_teaching_assistant import WanxTeachingAssistant

# 初始化
assistant = WanxTeachingAssistant(api_key="your_api_key")

# 生成教学视频
result = assistant.generate_lesson_video(
    subject="生物",
    topic="有丝分裂",
    grade="高中",
    style="animation",
    duration=25
)

# 获取结果
print(result.video_url)
print(result.storyboard)
print(result.teaching_notes)
```
