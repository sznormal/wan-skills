# 万相教学助手 - 参赛作品总结

## 作品亮点

### 1. 教育场景深度结合
专为教师设计，解决教学可视化内容制作的真实痛点

### 2. 技术创新
完整利用阿里通义万相Wan 2.7的三大核心能力：
- wan2.7-image-pro（文生图）
- wan2.7-t2v（文生视频）
- wan2.7-i2v（图生视频）

### 3. 可复用性强
不是一次性生成工具，而是可反复使用的完整教学素材生成系统

### 4. 功能完整
- 智能提示词优化
- 自动分镜脚本生成
- 批量素材生成
- 教学笔记自动输出

## 文件结构

```
万相教学助手/
├── SKILL.md                    # Skill核心说明文档

├── README.md                   # 快速上手指南

```

## 快速开始

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 设置API密钥
export DASHSCOPE_API_KEY="your_key"

# 3. 运行示例
python examples.py
```

## 核心功能演示

### 生成教学视频
```python
from main import WanxTeachingAssistant

assistant = WanxTeachingAssistant()
result = assistant.generate_lesson_video(
    subject="生物",
    topic="细胞分裂",
    grade="高中",
    style="知识动画风",
    duration=30
)
```

### 仅生成分镜
```python
storyboard = assistant.generate_storyboard_only(
    subject="历史",
    topic="丝绸之路",
    grade="初中"
)
print(storyboard["script"])
```
