# 万相教学助手 - 参赛作品说明书

## 作品信息

- **作品名称**: 万相教学助手（知识点动态可视化工具）
- **参赛赛道**: "万相妙思+"快闪赛
- **参赛主题**: 万相皆可Skill
- **参赛者**: Normal老师
- **参赛身份**: 教育工作者/教师

## 一、作品概述

### 1.1 核心定位

本Skill专为教育工作者设计，利用阿里通义万相Wan 2.7的多模态生成能力，将抽象的教学知识点快速转化为生动的动态视频和图片素材。实现"输入知识点 → 自动生成教学视频素材"的端到端工作流程。

### 1.2 核心价值

| 价值维度 | 具体描述 |
|---------|---------|
| **降本增效** | 无需专业视频制作技能，5分钟内完成知识点可视化 |
| **质量保证** | 专业级教学素材，支持4K画质输出 |
| **可重复使用** | 同一知识点可生成多种风格的教学素材 |
| **场景全覆盖** | 支持数理化生地语历全学科 |

### 1.3 解决的问题

- ❌ 教师缺乏视频制作技能
- ❌ 购买教学视频素材成本高
- ❌ 通用素材缺乏针对性
- ❌ 制作周期长、效率低

## 二、技术架构

本Skill充分利用阿里通义万相Wan 2.7的核心能力：

## 三、功能特性

### 3.1 学科全覆盖

| 学科 | 适用风格 | 示例知识点 |
|------|---------|-----------|
| 生物 | 知识动画风/自然写实风 | 细胞分裂、光合作用、DNA复制 |
| 物理 | 科幻科技风 | 电磁感应、牛顿定律、量子力学 |
| 化学 | 科幻科技风 | 化学反应、分子结构、周期表 |
| 历史 | 历史复古风 | 朝代更替、历史事件、人物故事 |
| 地理 | 自然写实风 | 地形地貌、气候类型、人口迁移 |
| 语文 | 历史复古风/艺术风格 | 诗词意境、小说场景、人物刻画 |
| 数学 | 简约线条风 | 几何图形、函数图像、证明过程 |

### 3.2 风格多样化

```
知识动画风 ──────▶ 简洁扁平，适合抽象概念
     │
科幻科技风 ──────▶ 未来感，适合科学实验
     │
历史复古风 ──────▶ 古典韵味，适合历史文学
     │
自然写实风 ──────▶ 真实自然，适合生物地理
     │
简约线条风 ──────▶ 极简清晰，适合数学逻辑
     │
多彩插画风 ──────▶ 活泼可爱，适合低年级
```

### 3.3 故事板模板

预置5种教学视频结构模板：

1. **概念讲解**: 引出 → 定义 → 案例 → 总结 → 思考
2. **实验演示**: 目的 → 准备 → 步骤 → 分析 → 结论
3. **历史叙事**: 背景 → 经过 → 人物 → 意义 → 启示
4. **文学赏析**: 引入 → 片段 → 手法 → 情感 → 推荐
5. **复习总结**: 框架 → 重点 → 易错 → 练习

## 四、使用示例

### 4.1 快速开始

```python
from main import WanxTeachingAssistant

# 初始化
assistant = WanxTeachingAssistant(api_key="your_api_key")

# 生成完整教学视频
result = assistant.generate_lesson_video(
    subject="生物",
    topic="有丝分裂",
    grade="高中",
    style="知识动画风",
    duration=30
)
```

### 4.2 快速生成教学图片

```python
# 仅生成配套图片
result = assistant.generate_quick_image(
    subject="物理",
    topic="电磁感应原理",
    grade="高中",
    style="科幻科技风"
)
```

### 4.3 仅生成分镜脚本

```python
# 不调用API，仅生成教学设计
storyboard = assistant.generate_storyboard_only(
    subject="历史",
    topic="丝绸之路",
    grade="初中",
    duration=45,
    template="历史叙事"
)
print(storyboard["script"])
```

### 6.2 自动分镜生成

```
【场景1】开场引入 (5秒)
  - 视觉：显微镜下的细胞画面
  - 动画：淡入 + 缩放
  - 教学：引入"分裂"概念

【场景2】核心讲解 (12秒)
  - 视觉：分裂过程分步展示
  - 动画：阶段切换 + 箭头标注
  - 教学：详细讲解四个阶段

【场景3】关键强调 (5秒)
  - 视觉：重点标记
  - 动画：闪烁 + 颜色变化
  - 教学：强调考试重点

【场景4】总结回顾 (8秒)
  - 视觉：思维导图式总结
  - 动画：依次呈现
  - 教学：巩固记忆
```
