# 🎨 万相Skill集合

> **参赛项目**：WaytoAGI整活计划第十二期「万相妙思+」快闪赛
> **主题**：「万相皆可Skill」
> **开发者**：Normal老师团队

## 📚 项目概述

本仓库汇集了基于阿里通义万相 Wan 2.7 API 开发的创意教育Skill集合，致力于让教育变得更有趣、更高效、更具传播力。

**核心理念**：让AI成为教育的创意伙伴，而非替代者。

---

## 🛠️ Skill列表

### 1. 📖 [万相教学助手](./万相教学助手/)
**让抽象知识变得可视化、可触摸**

- 15种教学可视化模板（思维导图、概念图谱、时间轴等）
- 智能知识点提取与可视化
- 一键生成教学素材

**适用场景**：备课素材、课堂演示、学生复习

---

### 2. 📜 [历史朋友圈生成器](./历史朋友圈生成器/)
**让历史人物"活"起来，用朋友圈方式学历史**

- 历史人物朋友圈模拟
- 重大事件"直播"体验
- 时代氛围沉浸式还原

**适用场景**：历史课堂导入、知识点讲解、社交媒体科普

---

### 3. 🔬 [科学梗图生成器](./科学梗图生成器/)
**把科学知识变成爆笑梗图**

- "当你终于理解相对论时"
- "物理老师眼中的我 vs 我眼中的物理"
- 病毒式知识传播

**适用场景**：课堂导入、社交媒体传播、趣味科普

---

## 🔧 技术栈

所有Skill基于以下技术：

- **核心模型**：阿里通义万相 Wan 2.7
  - `wan2.7-image-pro`：高质量图像生成
  - 文字渲染能力：中文文案支持
  - 风格控制：多种视觉风格

- **开发语言**：Python 3.x
- **API调用**：RESTful API

---

## 🚀 快速开始

### 环境配置

```bash
# 设置API Key
export WAN_API_KEY="your-api-key"

# 安装依赖
pip install requests
```

### 使用方式

每个Skill都是独立的模块，可以单独使用：

```bash
# 万相教学助手
cd 万相教学助手
python scripts/generate.py --concept "光合作用" --template "概念图谱"

# 历史朋友圈生成器
cd 历史朋友圈生成器
python scripts/generate.py --person "李白" --event "送孟浩然之广陵"

# 科学梗图生成器
cd 科学梗图生成器
python scripts/generate.py --concept "相对论" --type "当你终于理解时"
```

## 🌟 设计理念

### 教育为本
- 所有功能围绕教学场景设计
- 注重知识传递的有效性
- 降低教师使用门槛

### 创意为魂
- 用新颖形式激活学习兴趣
- 借鉴社交媒体传播模式
- 让知识变得"可分享"

### 技术为用
- 充分利用Wan 2.7的多模态能力
- 保持代码简洁可维护
- 支持二次开发和扩展
